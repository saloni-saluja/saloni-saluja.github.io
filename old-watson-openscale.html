IBM Watson OpenScale

The following content was the Getting Started page for IBM Watson OpenScale on Cloud Pak for Data as a Service. I performed a content audit and compiled a list of problems with this content.

Original content

Here's a list of problems that I found with the original content:

  • "Getting started tutorial" is not a good title. It does not contain the component name and it is not a tutorial.
  • It does not function as an overview page for Watson OpenScale as it lacks focus and tries to do too much.
  • The topic is a jumble of information without a clear point or organization.
  • It lacks a description of the UI.
  • Setup options are confusing and do not map to the UI experience.
  • The topic is wordy, and lacks modularity.
  • A link to the parent topic is missing.

New content

To fix the problems with the original content, I designed, documented, and published new documentation.
